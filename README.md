# Typescript React Demo ![CI status](https://img.shields.io/badge/build-passing-brightgreen.svg)

This is a demo application of using Typescript react application with redux.

## Installation
```
npm install
npm start
```
### Built With
* React 16.3.2
* Redux 4.0.0
* Typescript 2.8
* Redux-saga 0.16.0
* Semantic UI 2.3.1

### Requirements
* Nodejs
* npm
